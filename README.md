# Instagram風SNSアプリ（Laravel + Blade）

このアプリは、ITスクールのチーム開発として作成したInstagram風SNSです。
ユーザーが投稿を作成・編集・閲覧できる機能を持ち、Laravel Bradeを使ってフロントエンドを構築しています。

---

## 🌟 使用技術

- PHP 8
- Laravel 11
- Bladeテンプレート
- MySQL

---

## 🔧 主な機能

-  ユーザー登録・ログイン（Laravel UI）  
-  投稿作成フォーム & 複数枚の画像アップロード処理
-  プロフィール編集機能
-  フォロー/フォロワー機能  
-  管理者画面（Admin）
